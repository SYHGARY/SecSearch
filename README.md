# 🔍SecSearch🛡️
密文状态下高优化的支持精准加密以及模糊加密的数据库查询系统

## 后续计划（更新在方案设计中）
（1）使用OpenHiTLS库代替GmSSL，OpenHiTLS 的 SM4 算法实现性能更优，而SM4是轻量级分组密码算法，单条数据解密的计算耗时极短，多线程场景下的任务调度、队列同步、上下文切换的额外开销，很容易抵消并行计算带来的性能提升，甚至出现负优化。  
（2）补充完善密钥管理模块的方案设计，新增密钥轮换、密钥版本管理、密钥状态管控、历史密钥兼容等机制，覆盖密钥生成、存储、启用、轮换、停用的全生命周期管理。   
（3）扩充测试方案设计，补充异常输入、边界值、并发稳定性、性能基准、安全合规等多维度测试用例，构建完整的测试体系。
